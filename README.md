# FinSight

AI-Powered Expense Intelligence Platform

## Overview

FinSight is a Progressive Web Application (PWA) designed to help users manage and analyze their expenses intelligently.

## Core Features

* Expense Tracking
* OCR-Based Bill Scanning
* Bank Statement Analysis
* Dashboard Analytics
* Digital Signatures
* Document Management
* PWA Support

## Technology Stack

### Frontend

* React
* TypeScript
* Tailwind CSS

### Backend

* FastAPI

### Database

* PostgreSQL (Supabase)

### OCR

* Tesseract OCR
